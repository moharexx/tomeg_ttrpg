Chess

Requirements:

  • Skill: 200 Melee Weapon | 200 Martial Arts

Unlock Cost:

  • 200 XP

Tags:

  • Offense, Defense
  • Weapon Damage, Mobility
  • Main-Action / Continuous

Description

When using this Ability, you are put in Chess Stance.
While in Chess Stance you can perform different Tricks.
If you don't have the Stacks to Spend you cannot do the Trick.
These Tricks count as Abilities, unless said otherwise.
Max Stack: 6

  • While in Chess Stance if you Take Damage, you have to use the King Trick once, utill the End of Your Next Turn or the Stance Breaks.
  • If The Stance Breaks, you gain the Stagered Condition.

Pawn (Main-Action):

  • Gain 1 Stack, and use the Move-Action.
  • Using the Pawn Trick does NOT count as an Ability.

Knight (Main-Action):

  • Spend 1 Stack, then Teleport withing MS Range.
  • You need not Pinpoint the location, this can go over solid barriers, and does NOT Activate Reactions and Triggers.

Bishop (Free-Action):

  • Spend 2 Stacks, then use any Bonus-Action Action (Like Abilities) as a Free-Action.
  • You can only use this Once per Round.

Rook (Main-Action):

  • Spend 2 Stacks, then use the Move-Action and gain (6 × ESS) Armor VS All till the End of Your Next Turn.
  • Using the Rook Trick does NOT count as an Ability.

Queen (Main-Action):

  • Spend 3 Stack, then Teleport withing MS Range, the Next Action will gain (4 × ESS) Potency
  • The Teleport doesn't need Pinpointed location, this can go over solid barriers, and does NOT Activate Reactions and Triggers.
  • If the Potency Bonus is not used till the End of your Next Turn, it will be lost.

King (Main-Action):

  • Spend 1 Stack, then use either the Move-Action or the Attack-Action.
  • Using the King Trick to make a Move-Action does NOT count as an Ability.

The Stance Breaks when: Not Using the King Trick after taking Damage
