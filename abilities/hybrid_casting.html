Hybrid Casting

Requirements:

  • Innate: 5 INT, 5 WILL, 5 EGO

Unlock Cost:

  • 5 XP

Tags:

  • Utility
  • Buff, Soul Save

Description

Your CS can be INT + WILL + EGO, Capped at Total XP / 10
Your Soul Save Modifier can be (INT + WILL + EGO) / 2, Capped at Total XP / 20
When using the Memorization and Wishing feature you must still only use the specified Stat written!
