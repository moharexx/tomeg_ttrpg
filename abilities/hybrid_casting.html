Hybrid Casting

Requirements:

  • Innate: 5 INT, 5 WILL, 5 EGO

Unlock Cost:

  • 5 XP

Tags:

  • Utility
  • Spellcasting, Soul Save

Description

When Casting Spell Type Abilities, you can treat your CS to be (INT + WILL + EGO) up to a Maximum of (Total XP / 10)
When using the Memorization and wishing feature you still must use only the specified Stat written.

When making a Soul Save, you can treat the Modifier to be ((INT + WILL + EGO) / 2) up to a Maximum of (Total XP / 20)
