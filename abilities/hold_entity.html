Hold Entity

Requirements:

  • Spell: (20 INT | 20 WILL | 20 DES), (10 Enchantment | 20 Abjuration | 50 Transmutation)

Unlock Cost:

  • 20 XP

Tags:

  • Offense
  • Soul Save, Targeting, Crowd Control
  • Main-Action

Description

When used, you Paralyze a Pinpointed Entity or more.
You have Targeting Points equal to 6 × ESS, you can spend Targeting Points on Entities to Target them.
The Soul Save's Difficulty is equal to the Targeting Points spent on the Target.
On a Failed Save the Target gain the Paralyzed Condition.
Entities at the End of Their Turn can repeat the Soul Save, Ending the Paralyzed Condition on a success.

The Paralyzed Target is stiffened, but can be knocked over.

Note: Making a Save doesn't count as an Action if Action Cost isn't specified, so can be done while Paralyzed.
