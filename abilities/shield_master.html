Shield Master

Requirements:

  • Skill: 5 Melee Weapon

Unlock Cost:

  • 5 XP

Tags:

  • Defense
  • Self
  • Bonus-Action / Continuous

Description

When using this Ability, you are put in Dual Wielder Stance.
While in Dual Wielder Stance you gain access to 2 new Actions, which you can use only while Wielding a Shield.
These Actions don't Trigger Shield Properties.

Human Barricade (Main-Action + Bonus-Actions):

  • Chosen Sensed Entities within a Radius of ESS^(1/3) (you included) get the Defense-Action.
  • This Defense-Action ends at the End of Your Next Turn, or if they are not in the Radius.
  • This counts as a Skill Type Ability with the Defense Tag.

Tactical Block (Bonus-Action):

  • You make All Saves with Advantage until the End of Your Next Turn.
  • This counts as a Skill Type Ability with the Defense Tag.

The Stance Breaks only when entering any other Stance.
