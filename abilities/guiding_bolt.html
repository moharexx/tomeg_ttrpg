Description

You create an orb of Light, which you shoot at an Entity.
This is a Single Target Projectile which inflicts the Light Effect on Hit.

Light Effect:

  • The Target glows, revealing the Target in Darkness, and Invisibility.
  • The Next Attack against the Target has 5 × ESS Bonus To-Hit, and adds Bonus On-Hit Damage equal to this Ability's.
  • This Effect Ends at the Start of Your Next Turn, or when the Target is Hit.
