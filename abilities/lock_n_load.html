Lock'n Load

Requirements:

  • Skill: 10 Ballistic

Unlock Cost:

  • 5 XP

Tags:

  • Utility
  • Buff
  • Main-Action / Main-Action + Bonus-Action

Description

When using this Ability, you are put in Lock'n Load Stance.
While in Lock'n Load Stance, Slow-Reload's Action Cost is only a Bonus-Action.
While in Lock'n Load Stance, Reload's Action Cost is a Free-Action, only once a Turn.
When Reloading or Slow-Reloading, You can Reload all currently Wielded Weapons.

Stance Trigger

  • You instantly Reload and Slow-Reload any chosen Weapons in Natural Melee Reach.

This Stance Breaks only when entering another Stance.
