Call Lightning

Requirements:

  • Spell: (20 INT | 20 WILL | 20 EGO), (10 Evocation | 20 Conjuration | 50 Transmutation)

Unlock Cost:

  • 20 XP

Tags:

  • Offense
  • Electric Damage, Flow Sphere AOE
  • Main-Action + Bonus-Action / Free-Action

Description

When using this Ability, you are put in Call Lightning Stance.
For each time you Enter this Stance, you gain a 1 Stack.
The Range of this Stance Increases with Stacks.
The Lightning Bolts' Starting Positions can be offset by up to (10 × Stacks) × CS^(1/3) from the Caster, to a Pinpointed location.
Max Stacks Count: 3

Multiple Lightning:

  • Call down 1 + Stacks Seperate Lightning Bolts onto Different Targets.
  • Each Lightning is it's own Ranged Attack, Roll only a Single Attack for all of them, but seperate Damage.
  • Using On-Attacks will Apply it to All Lightning Bolts made this way.
  • Damage: (2d6 × ESS) Electric

Single Lightning:

  • Call down a Single Strong Bolt of Lightning, this is a Flow Sphere AOE Ranged Attack.
  • Damage: (2d6 × ESS) + (2 × Stacks × ESS) Electric
  • Radius: ESS^(1/3)

Electric Effect:

  • Against Soaked, or Airborn Targets this Ability Deals Max-Damage.

The Stance Breaks when: Using Abilities, Moving, HD Decrease, Taking Damage
