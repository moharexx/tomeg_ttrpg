Fog Cloud

Requirements:

  • Spell: (5 INT | 10 WILL | 10 EGO), (5 Conjuration | 10 Transmutation | 20 Enchantment)

Unlock Cost:

  • 5 XP

Tags:

  • Utility, Unlock
  • Obscurance, Static Sphere AOE
  • Main-Action

Description

When used, you create a cloud of fog, which Sight based detection cannot see through.

A Wind or Shockwave based Attack with at least 20 × ESS Damage disperses the Fog Cloud.
If that Attack is an AOE then 5 × ESS Damage is enough to disperse the Fog Cloud.

Shaped Cloud (50 Transmutation):

  • This Ability can be Shaped in whatever way the Caster desires, still fitting inside the Radius.

Be gone beholder!
