Description

When using this Ability, you are put in Deathblow Stance.
When Deathblow is Triggered, you make a Melee Attack.

Movement Impaired:

  • if the Target's MS is currently being Reduced, Double the Damage.

The Stance Breaks when: Using Abilities, Moving, HD Decrease

This Ability CANNOT CRIT!
