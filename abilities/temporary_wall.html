Description

When using this Ability, you are put in temporary wall Stance.
When you Enter, you are put in temporary wall Stance, you create a Wall made of a light pseudo material.
You decide the Wall's position, rotation, and if it's transparent or not.
The Wall can either be anchored to a position or it stays in the same position relative to you.
While in this Stance, the Stance's Cast Time is a Bonus-Action.
For Entities that try to Pass Through the wall, the wall has 20 × ESS STR.
The Wall has the same Pass Through Check as its Caster when the Caster uses Pass Through or when the Wall is trying to use it.
The Caster can Shove Entities with the Wall as A Bonus-Action.

The Stance Breaks only when entering any other Stance.
