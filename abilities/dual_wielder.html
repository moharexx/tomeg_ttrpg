Dual Wielder

Requirements:

  • Skill: 5 Melee Weapon

Unlock Cost:

  • 5 XP

Tags:

  • Offense, Defense
  • Self
  • Bonus-Action / Continuous

Description

When using this Ability, you are put in Dual Wielder Stance.
While in Dual Wielder Stance you gain access to 2 new Actions, which you can use only while Dual Wielding.

Blade Barrage (Main-Action):

  • You can make 3 Attack-Actions against 3 different Targets.
  • Even if the Attack-Action is somehow AOE, it still doesn't Target the same Entity more than once.
  • Roll only a single d10 Check for all of them, still Roll Damage separately.
  • You can also use a single Switch-Action after or before any of these Attacks.
  • This counts as a Skill Type Ability with the Offense Tag.

Blade Block (Bonus-Action):

  • You use the Defense-Action.
  • This counts as a Skill Type Ability with the Defense Tag.

The Stance Breaks only when entering any other Stance.
