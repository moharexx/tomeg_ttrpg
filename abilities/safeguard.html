Description

When the Ability is used, the Pinpointed Entity's Saving Throw will have a Bonus.
This Ability can be used after determiming if the Save is a Fail or not.

Potency Boost:

  • For each 2 points of Potency you can add +1 to the Save Bonus.
  • For each 3 × ESS points of Potency you can Target an additional Entity.

Note: This Ability cannot CRIT.
